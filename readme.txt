Place random quotes or trivia facts on your site through shortcodes or widgets

== Description ==

Factoid allows you to place interesting, amusing and inspiring little random 
facts and quotes on your WordPress website through both shortcodes (for inserting 
into posts and pages) and widgets (for inserting into sidebars and other layout 
elements). Factoid allows you to select the type and category of Factoid you'd like 
displayed, and also provides a number of display and layout options. Factoids are 
fetched across the internet from our extensive and constantly growing database.

== Installation ==

Install using the WordPress plugin installer. To manually install:

1. Create a folder called 'factoid' in the '/wp-contents/plugins/' directory
2. Upload the contents of the zip file to the 'factoid' folder
3. Activate the plugin through the 'Plugins' menu in WordPress
